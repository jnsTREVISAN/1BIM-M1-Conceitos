# 📘 Projeto: Conceitos de IoT e Sistemas Embarcados

## 📖 Sobre o projeto
Este trabalho foi desenvolvido como parte da disciplina, com o objetivo de compreender conceitos fundamentais de Internet das Coisas (IoT) e sistemas embarcados.

As atividades incluem questões conceituais, aplicações práticas, investigações em simuladores e reflexões sobre tecnologia.

---

## 🎯 Objetivos
- Entender o que é IoT
- Compreender o funcionamento de sistemas embarcados
- Aplicar conceitos em situações práticas
- Explorar simuladores como Wokwi e Tinkercad
- Desenvolver pensamento crítico sobre tecnologia

---

## 🛠️ Tecnologias utilizadas
- Markdown
- Simuladores (Wokwi / Tinkercad)
- Git e GitHub
